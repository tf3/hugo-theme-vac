# Hugo Theme Vac

Lightweight [Hugo](https://gohugo.io) blog theme. Demo at [vaccha.com](https://vaccha.com).

## Features

- Responsive
- Clean, minimal look
- Full tag support
- No JavaScript or third-party frameworks
- Table of contents listing on posts (optional)
- Google Analytics integration (optional)
